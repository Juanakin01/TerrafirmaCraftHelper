# TerrafirmaCraftHelper
Tools for anvil hit sequences and alloy calculations.  Requirements: Python 3.10+ 

It's designed for Hardrock V3. It's my first time playing TFC, so I don't know if the anvil works differently or if alloys change in other versions.

At least the alloy part can be easily edited from the JSON file included in the project. You can add or modify anything you want, the program will take it into account.
